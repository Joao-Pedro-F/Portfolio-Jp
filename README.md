## 🛠️ Tecnologias e Conceitos
O site foi construído utilizando as melhores práticas de front-end estático para garantir performance e acessibilidade:

* **HTML5 Semântico:** Melhor indexação e acessibilidade.
* **CSS3 Moderno:** Uso de Variáveis (Root), Flexbox e CSS Grid para um layout totalmente responsivo.
* **JavaScript (ES6+):** Renderização dinâmica de projetos através de objetos JSON.
* **Dark Mode:** Implementação de tema claro/escuro nativo.
* **DevIcons:** Integração de ícones das tecnologias utilizadas.

---

## 🎯 Destaques do Portfólio
O portfólio apresenta três projetos principais que demonstram meu domínio no desenvolvimento de sistemas:

1.  **Sistema de Gestão Escolar:** Foco em CRUD e modelagem de banco de dados PostgreSQL.
2.  **Sistema de Login:** Foco em segurança, autenticação e uso de Flask/MySQL.
3.  **API de Tarefas (To-Do):** Demonstração de arquitetura em camadas (Controller, Service, Repository) usando Spring Boot.
